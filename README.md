# Archer AI Project

射箭辅助工具 - 专业的射箭训练和计分应用

## 项目简介

这是一个基于 uni-app 框架开发的射箭辅助工具应用，支持跨平台部署（Web、Android、iOS）。应用提供计数器、计分系统、数据统计等功能，帮助射箭爱好者记录训练数据、计算成绩、提升训练效果。

## 功能特性

### ✅ 已完成功能

#### 计数器模块
- 箭数计数功能
- 自定义步长设置（1-50箭）
- 实时计时器
- 训练记录保存
- 日期范围筛选
- 训练趋势图表（折线图/柱状图）
- 分页显示（默认10条/页）

#### 计分模块
- 完整的计分系统
- 支持多种弓种（反曲弓、复合弓、传统弓）
- 支持多种距离（18米、30米、50米、70米、90米）
- 支持多种靶纸（122cm、80cm、60cm、40cm）
- 实时计分和计时
- 组别分数柱状图
- 详细记录展示
- 日期范围筛选
- 分页显示（默认10条/页）

#### 设置模块
- 深色模式切换
- 语言切换（简体中文/English）
- 消息通知设置
- 数据导出（JSON格式）
- 数据清除功能
- 关于信息

### ⏳ 待完成功能
- 数据统计分析
- 全局深色模式应用
- 多语言翻译完善

## 数据存储

应用使用 uni-app 的本地存储 API 进行数据持久化：

- **训练记录**: `training_sessions`
- **计分记录**: `scoring_sessions`
- **应用设置**: `settings`

### 数据管理策略

- 自动数据归档：超过100条记录时自动归档
- 定期数据清理：30天前的数据自动清理
- 分页加载：支持分页显示大量数据
- 存储监控：实时监控存储空间使用情况

详细文档请参考 [STORAGE_GUIDE.md](docs/STORAGE_GUIDE.md)
